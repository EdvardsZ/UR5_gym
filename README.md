# gym-UR5
Deep reinforcement learning environment in Ignition Gazebo for UR5 with RG2 gripper
## Setup

Install Ignition Gazebo from here.

Then clone this repo:
```
git clone https://github.com/EdvardsZ/UR5_gym.git
```
